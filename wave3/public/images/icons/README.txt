This folder contains desktop icon images.

The wastebin.png icon should be a 96x96 pixel image representing a trash bin.
You should add this image file to this directory.

Example sources:
1. Create your own icon
2. Download from a free icon repository (like Flaticon, IconFinder, etc.)
3. Use a placeholder service like: https://placehold.co/96x96?text=Trash
