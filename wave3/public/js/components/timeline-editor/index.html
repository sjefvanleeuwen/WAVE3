Timeline Editor Component

A customizable timeline editor component for the WAVE3 application.

Features

  • Drag and drop timeline items
  • Resize items from either edge
  • Multiple tracks support
  • Zoom in/out functionality
  • Customizable appearance
  • Time markers with proper formatting
  • Item collision detection
  • Grid snapping support

// Basic usage example
const timeline = document.createElement('timeline-editor');
timeline.setAttribute('duration', '120');  // 2 minutes
timeline.setAttribute('tracks', '4');      // 4 tracks
document.body.appendChild(timeline);

// Add an item
timeline.addItem({
    track: 0,
    start: 10,
    end: 25,
    title: 'Example Item',
    color: '#4285f4'
});

// Listen for changes
timeline.addEventListener('timeline-change', (e) => {
    console.log('Item changed:', e.detail.item);
});
